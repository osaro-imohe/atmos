## Installation and Local Setup

Run the command `git clone https://github.com/osaro-imohe/atmos` on your terminal to clone this repo to your current directory.

Run `yarn` to install all required dependencies.

Run `yarn start` to run the project locally.

Visit `localhost:3000/homes` or `localhost:3000/lots` to get started.
